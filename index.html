Guide Utilisateur (Terrain)

1. À la Voiture : Dès votre arrivée au stationnement, cliquez sur VÉHICULE. Votre point de retour est sécurisé.

2. Économie de Batterie : Vous pouvez maintenant basculer votre iPhone en Mode Avion. Le GPS restera actif et fonctionnera hors-ligne.

3. Lancer la marche : Cliquez sur PROSPECTION. Rangez le téléphone. Votre parcours se dessine automatiquement en arrière-plan tous les 5 mètres.

4. Analyse Sécheresse/Lidar : Sélectionnez une carte dans le menu Choix du temps, puis glissez le curseur horizontal pour ajuster la transparence.
